Los Media Query @media

Son unos elmentos añadidos a CSS tras la llegada de los smartphones que proporcionan unos comandos simples de programación para reponder a diferentes tamaños de pantalla. Anteriormente esto se hacía con javascript

Las consultas de medios se pueden usar para verificar muchas cosas, como:

  • ancho y alto de la ventana gráfica
  • ancho y alto del dispositivo
  • orientación (¿la tableta/teléfono está en modo horizontal o vertical?)
  • resolución
